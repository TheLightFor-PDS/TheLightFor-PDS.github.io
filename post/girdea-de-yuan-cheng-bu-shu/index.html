Girdea 的远程部署

（目前只介绍Github Pages的基础配置）

1.1 创建一个github账户

Github

（本部分不过多赘述，Sign up创建一个账户，建议用户名尽量短、好记）

1.2 创建github分配的域名仓库

[注：当你创建github账户的时候，官方会默认分配给你一个域名，也就是网站的网址

格式是：用户名.github.io，也就是说部署成功后，所有人都可以通过这个域名访问你的网站]

1.3 生成令牌

页面跳转后，复制Token，回到软件，选择远程

（单击保存后，先进行①，后进行②，如果显示同步成功，过几分钟后，即可用域名访问你的网站啦）

[注：如果检测远程连接错误，则删除仓库后，重新创建

​ 如果检测到同步错误，则重新获取令牌重试。ps：如果只是同步出错，多试几次即可，服务器不太稳定]
